Summary

  • SVG (scalable vector graphic) is graphic media format
  • You can enlarge or shrink SVG assets without compromising resolution
  • Vector graphics are digital images generated via mathematical formulas
  • SVG assets aid accessibility and SEO

Vector vs. Raster

There are The two general categories are raster (PNG, JPEG) and vector (SVG). Raster images include files that end in .png, .jpg, and .jpeg. Files that end in .svg are vector images.

Raster graphics have a tendency to pixelate if enlarged beyond their resolution capabilities. In contrast, vector media can shrink or enlarge withouth compromising resolution.

SVG hasn't quite hit "widespread adoption", more designers are discovering its benefits and integrating it into their web projects.

Originally, browser support for SVG was limited.

Perhaps this is why we haven't seen its widespread adoption

As browser support continues to expand, we can expect to see more designers seem to be using SVG to create digital assets like logos, animations, user interface elements, and icons.

Accessibility & SEO

Besides adding a polished look to your website, SVG supports accessibility and SEO. How? SVG files are text based which allows them to be searched and indexed by sites like Google, Bing, and Yahoo in addition to working with screen readers.

Tutorial: Make Some Shapes With SVG

This article covers some simple shapes you can build with SVG.

1. Circles

You need three attributes to code a circle with SVG:
r The radius of the circle.
cx The x position of the center of the circle.
cy The y position of the center of the circle.

2. Lines

3. Rectangles

x
The x coordinate of the rect.
Value type: <length>|<percentage> ; Default value: 0; Animatable: yes
y
The y coordinate of the rect.
Value type: <length>|<percentage> ; Default value: 0; Animatable: yes
width
The width of the rect.
Value type: auto|<length>|<percentage> ; Default value: auto; Animatable: yes
height
The height of the rect.
Value type: auto|<length>|<percentage> ; Default value: auto; Animatable: yes
rx
The horizontal corner radius of the rect. Defaults to ry if it is specified.
Value type: auto|<length>|<percentage> ; Default value: auto; Animatable: yes
ry
The vertical corner radius of the rect. Defaults to rx if it is specified.
Value type: auto|<length>|<percentage> ; Default value: auto; Animatable: yes
pathLength
The total length of the rectangle's perimeter, in user units.
Value type: <number> ; Default value: none; Animatable: yes
